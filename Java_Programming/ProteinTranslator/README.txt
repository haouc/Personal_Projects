This is a fun protein translator program.

Come with a sample protein sequence as a table.

Output a csv file with translated protein sequence with full amino acid names.

To compile this program (in terminal):
javac *.java

To run this program (in terminal):
java Main Protein.txt 0 2 5 700    (Protein.txt can be replaced by any txt files, the numbers mean required columns or rows to output in console.)
